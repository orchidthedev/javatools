# javatools
a collection of java based tools. currently contains a directory scanner, but may have new tools added on at a later time.

To use, unzip file and from terminal run `javac src/tools/Main.java` as the highest directory, "javatools"
then run `java src.tools.Main`

The scanner has a few features. You can scan by name or run a caps-insensetive scan, it also saves the directory when closed. You can select a directory by clicking the "Select Directory" button and change the theme by clicking the "Theme" button. Upon closing the theme changing menu, the main app will reload to update the theme.
